Présentation

{{SITE_NAME}} est une plate-forme anonyme de partage de contenus, images ou simples textes.
Le site ne requiert pas de cŕeation de compte pour participer. Il suffit en effet de créer un nouveau sujet via la page {% trans "participate" %}, ou de répondre à une quelconque discussion.

Participer

On distingue deux types de messages: les sujets et les réponses. Les deux sont composés de la même manière: une image et un texte descriptif (l'un ou l'autre peut parfaitement être omis).
Ne nécessitant pas de compte utilisateur, le site vous demandera pour chaque message d'entrer un mot de passe: celui-ci servira à pouvoir par la suite éditer ce message ou le supprimer.

Markdown

Vous pouvez utiliser la syntaxe Markdown lors de la rédaction de vos messages. Vous trouvez ici la liste des balises disponibles.

Anonymat

Toute l'application est parfaitement anonyme, aucune donnée permettant d'identifier l'utilisateur n'est enregistrée.
Une seule chose, pour faciliter les discussions, un nom d'auteur est attribué à chaque message. Ce nom est généré en fonction du sujet en cours, et du mot de passe entré. C'est à dire que si vous utilisez toujours le même mot de passe au sein d'une même discussion, vos messages apparaîtront sous le même nom d'auteur.
