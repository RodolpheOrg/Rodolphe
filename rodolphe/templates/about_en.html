Presentation

{{SITE_NAME}} is an anonymous platform of content sharing, pictures or simple texts.
The website does not require any account to participate. You can indeed create a new thread via {% trans "participate" %}, or reply to any discussion.

Participate

We distinguish two types of posts: threads and replies. Both are composed of a picture and a descriptive text (one or the other may be omitted).
Not needing any account, the website will ask you a password each time you write a message: this password let you edit or delete the post later.

Markdown

You can use Markdown syntax when redacting messages, you can see here the list of available tags.

Anonymity

The application is totally anonymous, no data about users are saved.
Just one thing, to make discussions easier, an author name is assigned to each post. This name is generated with the current thread and the password you enter. Thus, if you use the same password in a same thread, you will always get the same author name.
